🏨 Hotel Reservation System

Console-based Application using Java, JDBC, and MySQL

📖 Description

The Hotel Reservation System is a backend-focused Java console application that enables hotel staff to manage room reservations with ease. It uses JDBC for database operations and supports all basic CRUD functionality.

🚀 Features

  • 📥 Reserve a room with guest details
  • 📋 View all reservations in a clean table format
  • 🔍 Retrieve a room number by guest ID and name
  • ✏️ Update reservation records
  • 🗑️ Delete reservations securely

🛠 Technologies Used

  • Java
  • JDBC (Java Database Connectivity)
  • MySQL (Database)

🧰 Tools

  • Eclipse IDE / IntelliJ IDEA
  • XAMPP / MySQL Workbench

💻 Project Code Snippet & Demo

Here is a sample code snippet for inserting reservation data using JDBC:


// Java JDBC Insert Example
Connection con = DriverManager.getConnection("jdbc:mysql://localhost:3306/hotel", "root", "password");
PreparedStatement ps = con.prepareStatement("INSERT INTO reservations (guest_name, room_no, checkin, checkout) VALUES (?, ?, ?, ?)");
ps.setString(1, "John Doe");
ps.setInt(2, 101);
ps.setDate(3, Date.valueOf("2025-06-20"));
ps.setDate(4, Date.valueOf("2025-06-25"));
int result = ps.executeUpdate();
if (result > 0) {
    System.out.println("Reservation successful!");
}

You can test this on a local MySQL setup by creating the relevant table and running the console application.
